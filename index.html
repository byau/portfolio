This project is based on the traditional game of tic-tac-toe. Like tradition tic-tac-toe, this game is turn-based game where user (player X) and a friend (player O) will alternate to place a marker on the game board in attempt to have 3 markers of the same kind in a row, column or diagonal to win. If all the space on the board has been populated and no winners has emerge, the game will result in a tie.
